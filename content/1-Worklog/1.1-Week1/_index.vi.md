## Tổng quan
Tuần 1 tập trung vào các khái niệm cơ bản của AWS và quản lý tài khoản. Tuần này bao gồm thiết lập Tài khoản AWS, Budgets, Kế hoạch Support, và các kiến thức cơ bản về IAM.

## Danh sách công việc

### Ngày 2: Thiết lập & Khám phá Tài khoản AWS
- Tạo và xác minh Tài khoản AWS
- Cấu hình cài đặt AWS Console
- Thiết lập Xác thực Đa yếu tố (MFA)
- Hiểu về Cơ sở hạ tầng Toàn cầu AWS

### Ngày 3: Quản lý Hóa đơn & Chi phí
- Tạo và cấu hình AWS Budgets
- Thiết lập Cost Allocation Tags
- Kích hoạt AWS Cost Explorer
- Giám sát cảnh báo hóa đơn

### Ngày 4: AWS Support Plan & Tài liệu
- Xem xét các Kế hoạch AWS Support
- Chọn mức support phù hợp
- Hiểu về Service Level Agreements (SLA)
- Khám phá AWS Documentation

### Ngày 5: Nền tảng IAM
- Tạo IAM Users và Groups
- Cấu hình IAM Policies
- Hiểu về Role-Based Access Control (RBAC)
- Tạo và quản lý access keys

### Ngày 6: IAM Nâng cao & Best Practices Bảo mật
- Triển khai password policies
- Thiết lập MFA cho IAM users
- Tạo và quản lý IAM Roles
- Xem xét AWS Security Best Practices

## Thành tích đạt được
✅ Thiết lập thành công Tài khoản AWS với Multi-Factor Authentication  
✅ Cấu hình AWS Budgets với cảnh báo ngưỡng chi phí  
✅ Triển khai quản lý IAM users với mô hình least-privilege access  
✅ Tạo baseline security policies tuân theo nguyên tắc AWS Well-Architected  
✅ Ghi chép cấu trúc tài khoản AWS và tổ chức phân cấp  

## Tài nguyên sử dụng
- AWS Management Console
- AWS IAM Best Practices Guide
- AWS Budgets Configuration
- AWS Documentation Portal

## Ghi chú
- MFA là yếu tố quan trọng để bảo mật tài khoản
- IAM policies nên tuân theo nguyên tắc least privilege
- Theo dõi hóa đơn định kỳ để tránh chi phí bất ngờ

### Kết quả đạt được tuần 1:

* Tạo tài khoản AWS Free Tier thành công với các tiêu chuẩn bảo mật root account.
* Hiểu AWS Global Infrastructure: Regions, Availability Zones (AZs), Edge Locations để triển khai dịch vụ tối ưu.
* Cấu hình IAM với quyền hạn chi tiết: tạo IAM users với least-privilege policies, kích hoạt MFA.
* Cài đặt AWS Budgets và Cost Explorer để giám sát Free Tier và ngăn chặn phí bất ngờ.
* Cài đặt và cấu hình AWS CLI với credentials và region mặc định.
* Đánh giá AWS Support Plans và chọn support tier phù hợp cho phát triển.
* Thiết lập nền tảng quản trị tài khoản để vận hành cloud an toàn và tiết kiệm.
